# Security policy

Не публикуйте уязвимости, токены или приватные списки доменов в Issues. Для
сообщения об уязвимости используйте GitHub Private Vulnerability Reporting в
разделе Security репозитория.

Артефакты следует скачивать только из Releases `ldkt/happ-routing` и проверять
по `SHA256SUMS`. Команда проекта не контролирует содержимое внешних списков;
каждый release содержит `release.json` с происхождением данных.
